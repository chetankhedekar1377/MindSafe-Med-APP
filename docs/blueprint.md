# **App Name**: Clarity Care

## Core Features:

- Symptom Tracker: Allow users to log their symptoms with date and severity level.
- Medication Reminder: Send notifications to remind users to take their medications on time.
- Appointment Scheduling: Enable users to schedule and manage appointments with healthcare providers.
- Health Insights: Provide personalized health insights based on tracked data using a generative AI tool to correlate symptoms, medications, and appointments, offering potential explanations and suggesting questions for the next doctor's visit.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to promote calmness and trust.
- Background color: Very light blue (#F0F8FF) to provide a clean and unobtrusive backdrop.
- Accent color: Muted green (#A9DFBF) to signal important actions and information, inspired by nature.
- Body and headline font: 'Inter', a sans-serif font, to provide a modern, machined, objective, neutral look.
- Use minimalist icons inspired by Material Design, with rounded edges for a friendly feel.
- Maintain a clean and spacious layout with plenty of whitespace to reduce cognitive load.
- Employ subtle animations via Framer Motion to create smooth transitions and micro-interactions, enhancing user experience without causing distraction.
